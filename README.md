## Usage
Features:
* filter.KalmanFilter                 --> KF and EKF implementation
* filter.UnscentedKalmanFilter        --> UKF and SQ-UKF implementation
* filter.ParticleFilter               --> PF implementation
* filter.SquareRootInformationFilter  --> SR-IF implementation
